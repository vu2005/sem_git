• git init - Khởi tạo một repository Git mới
  • git clone <repository-url> - Tải về một repository từ URL
  • git add <file> - Thêm file vào stage
  • git commit -m "<message>" - Lưu thay đổi vào repository với thông điệp
  • git status - Kiểm tra trạng thái của repository
  • git pull - Lấy thay đổi mới nhất từ remote repository
  • git push - Đẩy các thay đổi lên remote repository
  • git branch - Liệt kê, tạo hoặc xóa nhánh
  • git checkout <branch-name> - Chuyển đổi sang nhánh khác
  • git merge <branch-name> - Hợp nhất một nhánh vào nhánh hiện tại
  • git log - Hiển thị lịch sử commit
  • git reset --hard <commit> - Khôi phục về commit cụ thể
  • git revert <commit> - Tạo commit đảo ngược các thay đổi
  • git stash - Lưu thay đổi tạm thời chưa commit
  • git remote - Quản lý các repository từ xa
  • git fetch - Lấy dữ liệu từ remote repository
  • git diff - So sánh các thay đổi
  • git rm <file> - Xóa file khỏi working directory và stage
